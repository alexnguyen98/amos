# Getting started

### SETUP

Installing npm

```bash
npm i
```

When adding an icon, generate a new icon index file

```bash
npm run generate-icon
```

When adding a component, generate a new component index file

```bash
npm run generate-component
```

### DEMO

Running the demo

```bash
npm run storybook
```
